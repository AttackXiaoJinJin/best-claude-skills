## Skills

| Skill | Description | Install |
|---|---|---|
| [**Skill Guide**](./skills/skill-guide) | Meta-skill for creating high-quality Claude Code skills — covers architecture, trigger design, and best practices | `npx skills add AttackXiaoJinJin/best-claude-skills --path skills/skill-guide` |

## Quick Start

Install any skill with:

```shell
npx skills add AttackXiaoJinJin/best-claude-skills --path skills/<skill-name>
```

Then invoke in your agent terminal:

```shell
/skill-guide    # Create or review a Claude Code skill
```
